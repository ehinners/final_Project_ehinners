Uber Tracker

An uber tracker is a timer that helps a player track, and predict, when medics will have uber charge. This tool also indicates which team has an uber ADVANTAGE, and how much advantage that team has. To learn more about medic, or what uber is, visit the Medic page. This app should default to common values found on most 5 capture point maps, simply press start when the round begins.

How to Use

The user can set initial ubercharge values, either with the sliders, or the numerical fields, and can initialize a representation of the gameclock. The reset button sets an individual charge to zero. NOTE: Because spawn times vary wildly depending on gamestate, gamemode, AND map, it's impossible for this calculator to delay uber built while a medic is dead. To utilize the reset button correctly, press it when the enemy RESPANWS. This reset button can be used WHILE the timer is running, OR paused. The start/stop button will either pause or start the running of the timer.

The kritz indicators, the percentage field on the right, and the arrows are un-interactable. The charge will always increase at the rate of the stock medigun, but at the point where if a medic would have full charge IF they were using kritzkrieg, the kritz indicator will light up. This is because the kritzkrieg charges faster than a standard medigun. To learn more about the kritzkrieg, visit the "weapons to use" section of the medic page.

The percentage indicator shows the DIFFERENCE between the two charge levels, and the arrows indicate which team has the larger charge.
